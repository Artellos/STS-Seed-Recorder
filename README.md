# STS Seed Recorder

> **Vibecoded** ‚Äî this project was built with vibes, caffeine, and AI assistance.

A webapp for recording and visualising all map nodes in **Slay the Spire 2** for a given run seed.

Maps can be generated instantly from a seed string using a pure-Python port of the STS2 map generation algorithm ‚Äî no game installation required.

---

## Features

- **Seed management** ‚Äî create, name, and delete seeds
- **Auto-generate maps** ‚Äî derive all 3 act maps from a seed in milliseconds (pure Python, no STS2 install needed)
- **Manual map editor** ‚Äî click to place/edit nodes, draw connections, mark your chosen path
- **Node detail panel** ‚Äî add notes to any node
- **Act tabs** ‚Äî switch between Act 1, 2, and 3 views
- **Dark STS-themed UI**
- Persistent SQLite storage ‚Äî no external database needed

## Map Generation

Maps are generated by `backend/map_generator.py`, a faithful Python port of the decompiled STS2 C# source (`MegaCrit.Sts2.Core`).

### Seed ‚Üí Map pipeline

1. `seed_string` ‚Üí `GetDeterministicHashCode(seed_string)` ‚Üí `uint base_seed`
2. For each act *N*: `map_rng = Rng(base_seed, "act_N_map")`
3. `StandardActMap` algorithm:
   - **Path generation** ‚Äî 7 iterations of random-walk from row 1 to the boss row, no crossing paths
   - **Type assignment** ‚Äî guaranteed rows (RestSite at top, Treasure/Elite at row‚àí7, Monster at row 1), then queue-based assignment with adjacency constraints
   - **Pruning** ‚Äî duplicate path segments removed
   - **Post-processing** ‚Äî center grid, spread adjacent nodes, straighten paths

### Act configurations

| Act | Rooms | Rest count |
|-----|-------|------------|
| Overgrowth (Act 1) | 15 | Gaussian(7, œÉ1, 6‚Äì7) |
| Underdocks (Alt Act 1) | 15 | Gaussian(7, œÉ1, 6‚Äì7) |
| Hive (Act 2) | 14 | Gaussian(6, œÉ1, 6‚Äì7) |
| Glory (Act 3) | 13 | Uniform(5‚Äì6) |

### Node types

`Ancient` ¬∑ `Monster` ¬∑ `Elite` ¬∑ `Treasure` ¬∑ `RestSite` ¬∑ `Shop` ¬∑ `Unknown` ¬∑ `Boss`

## REST API

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/seeds` | List all seeds |
| `POST` | `/api/seeds` | Create seed (`seed_value`, `name`) |
| `GET` | `/api/seeds/:id` | Get seed with nodes & connections |
| `PUT` | `/api/seeds/:id` | Update seed |
| `DELETE` | `/api/seeds/:id` | Delete seed |
| `POST` | `/api/seeds/:id/generate` | Generate all act maps from seed |
| `GET` | `/api/seeds/:id/generate/status` | Poll generation status (legacy) |
| `GET` | `/api/sts2/status` | Check generator availability |
| `POST` | `/api/seeds/:id/nodes` | Add a node manually |
| `PUT` | `/api/nodes/:id` | Update node (type, notes, on_path) |
| `DELETE` | `/api/nodes/:id` | Delete node |
| `POST` | `/api/seeds/:id/connections` | Add a connection |
| `DELETE` | `/api/connections/:id` | Delete connection |
